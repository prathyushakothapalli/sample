  9. What is inheritance in Python?

    A way to create multiple instances of a class.
    A technique to reuse code and create a new class based on an existing class.
    A method for accessing private members of a class.
    A mechanism for implementing polymorphism in Python

    10. See more

    Answer:Option B

    Explanation:Inheritance in Python allows a new class (subclass) to inherit attributes and methods from an existing class (superclass). It promotes code reuse and enables the creation of hierarchical relationships between classes

  10. Which keyword is used to inherit a class in Python?

    extends
    inherits
    sub
    class

    11. See more

    Answer:Option A

    Explanation:In Python, the extends keyword is used to inherit a class. For example, class SubClass(ParentClass): indicates that SubClass inherits from ParentClass.

  11. What is the method resolution order (MRO) in Python?

    The order in which methods are defined within a class.
    The order in which methods are executed within a program
    The order in which base classes are searched to resolve methods in a subclass.
    The order in which classes are instantiated in a program.

    12. See more

    Answer:Option C

    Explanation: Method Resolution Order (MRO) in Python defines the order in which base classes are searched to resolve methods and attributes in a subclass. It follows the C3 linearization algorithm to determine the MRO.
